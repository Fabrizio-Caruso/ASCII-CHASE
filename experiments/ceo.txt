CROSS CHASE/ASCII CHASE
Le but de ce jeu, c'est le plaisir de coder en découvrant les détails
techniques des ordinateurs des années 80.
Le code est écrit en ANSI C pour permettre d'avoir un jeu sur beaucoup
d'ordinateurs 8 bit en utilisant des cross-compilateurs comme
CC65 (pour les CPU 6502 et similaires) et ZZ88Dk (pour les Z80).
La base commune du code est agnostique des détails graphiques. Ce noyau, 
sans modifications significatives, nous donne un jeu avec des caractères ASCII/PETSCII/ATASCII.
Avec des patches spécifiques pour chaque version je peux rajouter des éléments graphiques et sonores
(actuellement c'est le cas pour les versions Oric, Commodore 64 et Comomdore 16/116/+4). 
Le jeu pourrait aussi supporter des vrais sprites à la place de caractères sans modifications
du noyau du code.
